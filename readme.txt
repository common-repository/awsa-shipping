روش های حمل و نقل با تنظیمات پیشرفته
== Description ==
این افزونه روش های ارسال پست پیشتاز، پست سفارشی ، تحویل اکسپرس و پست هوایی را به لیست روش های حمل نقل در ووکامرس و دکان با تنظیمات مختلف در هر روش اضافه می کند.

امکان ثبت کد رهگیری با قابلیت نمایش در بخش جزئیات سفارش ووکامرس
امکان ایمپورت و اکسپورت کد رهگیری سفارشات با اکسل

روش های ارسال
1. پست پیشتاز
2. پست سفارشی
3. تحویل اکسپرس
4. تحویل هوایی

تنظیمات روش های ارسال
1.  عنوان و تصویر دلخواه برای هر روش ارسال
2. مشخص کردن شهر های فعال و غیر فعال
3. حداکثر وزن مجاز
4. هزینه پیشفرض برای ارسال مرسوله
5. هزینه های اضافی برای درست نبود کد پستی به صورت درصدی یا مبلغ ثابت
6. هزینه ی اضافی به ازای بسته بندی و... به صورت درصدی یا مبلغ ثابت
7. تنظیمات ارسال رایگان براساس روش پرداخت , مجموع حداقل مبلغ سفارش برای ارسال رایگان
8. ارسال رایگان برای اولین سفارش مشتری

محاسبه ی هزینه ی پست پیشتاز و پست سفارشی براساس تعرفه ی ۱۳۹۹ محاسبه می شود.


== Installation ==

1. Download the plugin and extract the files
2. Upload `awsa-shipping` to your `~/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
